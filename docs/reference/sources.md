## Point Cloud Processing

| Resource | Description |
|----------|-------------|
| [3D Forest](https://www.3dforest.eu/) | Open-source tool for terrestrial LiDAR forest data segmentation and measurement. |
| [TreeLearn (Ecker Lab)](https://github.com/ecker-lab/TreeLearn) | Python package for automated tree segmentation and classification from point clouds. |
| [3DFin](https://github.com/3DFin/3DFin) | Toolkit for individual tree segmentation and feature extraction from LiDAR data. |
| [Computree](https://github.com/computree/computree-v3) | Modular open-source platform for 3D point cloud processing, including forestry plugins. |
| [lidR](https://github.com/ForestTools/lidR) | R package for airborne LiDAR data manipulation and forest metrics extraction. |
| [PDAL](https://pdal.io/) | Industry-standard point cloud data processing library (C++/Python). |
| [CloudCompare](https://www.cloudcompare.org/) | Powerful open-source 3D point cloud and mesh processing software. |
